# .dotfiles

### Description

My dotfiles for personal use.

##

### Installation

#### 1.) Clone repository

```shell
git clone https://github.com/ImBadAtPickinUsernames/.dotfiles.git
```
#### 2.) Make script executable
```shell
sudo chmod +x $HOME/.dotfiles/install.sh
```
#### 3.) Start script
```shell
$HOME/.dotfiles/install.sh

```
